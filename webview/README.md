
# BMW_MD Webview

This directory contains the webview files for BMW_MD WhatsApp bot.

## Usage

1. Put your HTML, CSS, and JavaScript files in this directory.
2. Access them through the bot using the `webview` command.
3. Example: `.webview dashboard` would open the dashboard page.

## Creating Custom Pages

Create additional HTML files in this directory to add new pages:
- `dashboard.html` - accessed via `.webview dashboard`
- `profile.html` - accessed via `.webview profile`
- etc.

You can also use URL parameters from the webview command to dynamically load content.
